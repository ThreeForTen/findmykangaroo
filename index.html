🦘 Know a kangaroo we're missing?

Our location data and fun facts are stored as simple JSON files that anyone can edit. You can help by submitting a pull request on GitHub:

📍 Edit locations.json — add a zoo, sanctuary, or wild habitat
🧠 Edit facts.json — add a fun kangaroo fact

Each location needs: name, lat, lng, type (wild/zoo), country, desc, and url (a link proving kangaroos are there). See the README for full details.
